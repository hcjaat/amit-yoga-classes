# Amit's Yoga Classes — Website

A modern, single-page yoga studio website built with **Next.js** and deployed on **Netlify**.

> 🧘 This is a client project — a professional website built **for** Amit Sir (Government Certified Yoga Instructor) by [Hemant Choudhary](https://github.com/hcjaat).

## 🧘 About

Professional website for **AMIT'S YOGA CLASSES** — a government-certified yoga studio based in Mahesh Nagar, Jaipur, Rajasthan. Features include gallery carousel, animated counters, testimonial slider, FAQ accordion, WhatsApp contact form, Google Maps integration, and responsive mobile navigation.

## ✨ Features

- Responsive design (mobile-first, hamburger menu)
- Typewriter hero animation with parallax background
- Auto-advancing gallery with lightbox modal
- Animated statistics counters
- Testimonial carousel with star ratings
- FAQ accordion
- WhatsApp contact form integration
- Google Maps location
- SEO optimized with JSON-LD structured data
- Google Analytics integration
- PWA support with manifest.json
- WebP optimized images
- 3D tilt animations on icons

## 🛠️ Built With

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Geist (sans), Playfair Display (serif)
- **Deployment:** Netlify

## 🚀 Getting Started

```bash
# Install dependencies
pnpm install

# Development server
pnpm dev

# Build for production
pnpm build

# Static export (for Netlify folder upload)
npx next build
# Output goes to `out/` folder
```
